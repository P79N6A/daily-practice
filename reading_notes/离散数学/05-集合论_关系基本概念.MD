# 第四周：集合论：关系基本概念

## 关系合成运算(composition)
R为A到B的二元关系，R⊆A×B

S为B到C的二元关系，S⊆B×C

R和S的合成关系**RoS**定义为：
```
RoS={<x,z>|x∈A∧z∈C∧∃y(y∈B∧xRy∧ySz)}
```
简化形式： 
```
RoS={<x,z>|∃y(xRy∧ySz)}
```

## 用关系矩阵表示合成运算
关系合成运算对应**关系矩阵**的**乘法**

将**数乘**换成**合取**，将**数加**换成**析取**
```
设|A|=m，|B|=n，|C|=p，
R⊆A×B，S⊆B×C，
则 MR=[rⁱʲ]m*n， MS=[sⁱʲ]n*p
T=RoS，有T⊆A×C，
MT=Mᴿ*Mˢ=[tⁱʲ]m*p
其中tⁱʲ=∨ᵏ=1..n(rⁱᵏ∧sᵏʲ)（i=1,..,m; j=1,..,p）
```

## 基数
设集合 _A, B_ 分别含有 _m, n_ 个不同的元素，则 _A_ 到 _B_ 的二元关系的个数为 _2ᵐⁿ_ 个。

设有限集合A, |A| = n, 则关系 |ρ(A×A)| = 2ⁿⁿ， 即 2 的 n² 个。
