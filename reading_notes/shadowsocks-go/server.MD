# 服务端实现

## 请求体

最长 269 字节：
```
1(addrType) + 1(lenByte) + 255(max length address) + 2(port) + 10(hmac-sha1)
```

### 请求体布局

| 部分      | 长度 (字节)  | 解释                             |
|:---------|:------------|:--------------------------------|
| 地址类型   | 1           | 1: IPv4; 3: 域名; 4: IPV6       |
| 地址长度   | 1           | 域名的长度 (如为IP4/6，则无此字段)  |
| 地址      | 变长，最大255 | 域名长度为上一字段规定，IP4/6定长   |
| 端口      | 2           |                                 |
| 哈希校验码 | 10          | HmacSha1(对以上字段的字节slice)    |
