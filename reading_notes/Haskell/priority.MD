# Haskell 算符优先级

  ```haskell
  0 $
  1 >>> <<< ^<< >>^ <<^ ^>> >>= =<< >=> <=< &
  2 ||| +++
  3 *** &&& <|>
  4 <*> <* *> <$> <$ $> <$$> <**>
  5 <+> ++
  6 <> + -
  7 % / *
  9 .
  ```

**```$``` 的作用在于把左边和右边的表达式都加上括号**

和 ```$``` 一样，```.``` 是另一个常用函数，其定义为：
```haskell
(.) :: (b -> c) -> (a -> b) -> a -> c
f . g = \x -> f $ g x
```
